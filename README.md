git clone https://github.com/WesanChien/Android_Socket_Chat.git

使用 Android Studio 開啟專案資料夾

選擇模擬器或實體 Android 裝置

點擊 Run 執行專案

#專案介紹

本專案為一個使用 Java 開發的 Android 即時聊天應用程式，
透過 Socket (TCP) 建立 Client 與 Server 之間的連線，
實作基本的即時文字訊息傳送與接收功能。

#功能說明

Client / Server Socket 連線

即時文字訊息傳送

聊天紀錄顯示

使用者名稱設定

連線與中斷連線控制

#執行環境

Android Studio

Java

Android SDK 31 以上

Android 裝置或模擬器

#備註
Client 與 Server 需位於同一區域網路
